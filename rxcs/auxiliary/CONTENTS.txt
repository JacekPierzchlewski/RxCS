RXCS toolbox, rxcs\auxiliary\ - directory with auxiliary modules

contents:

    1. krls\                    - directory with kernel recursive least squares modules

    2. CONTENTS.txt             - the current file

    3. Makefile                 - local Makefile

    4. __init__.py              - module initialization file


    5. L2Solv_aldkrls.py        - L2 solver which uses KRLS algorithm

    6. @L2Solv_aldkrls_test.py  - link to a module with tests for the L2 solver which uses kernel recursive least squares (KRLS) method

    7. irls.py                  - module with iterative reweighted least squares (IRLS) algorithms
