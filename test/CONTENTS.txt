RXCS toolbox, test\ - directory with the tests for RxCS toolbox modules

contents:

    1. rxcs~                   - soft link to the directory with all the RxCS modules

    2. CONTENTS.txt            - this file

    3. L2solv_aldkrls_test.py  - tests for the L2 solver which uses Kernel Recursive Least Squares Method

    4. Makefile                - local Makefile

    5. RxCSobject_tests.py     - tests for main base class for all modules of RxCS

    6. SNR_test.py             - tests for SNR evaluation of the reconstructed signal

    7. nonuniANGIE_test.py     - tests for nonuniform sampler with ANGIE sampling scheme

    8. sigRandMult_test.py     - tests for random multitone signal generator
