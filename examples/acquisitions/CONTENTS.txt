RXCS toolbox, examples\acquisitions\ - directory with examples on how to use
                                       the acquisition systems

contents:

    1. rxcs~                    - soft link to the directory with all the RxCS modules

    2. CONTENTS.txt             - this file

    3. Makefile                 - local Makefile


    # Nonuniform sampler examples (ANGIE sampling scheme):

    4. nonuniANGIE_ex0.py   - example #0 on how to use the ANGIE sampler - the simplest example

    5. nonuniANGIE_ex1.py   - example #0 on how to use the ANGIE sampler


    # Nonuniform sampler examples (ARS sampling scheme):

    6. nonuniARS_ex0.py     - example #0 on how to use the ARS sampler - the simplest example


    # Nonuniform sampler examples (JS sampling scheme):

    7. nonuniJS_ex0.py      - example #0 on how to use the JS sampler - the simplest example


    # Nonuniform sampler examples (external sampling scheme):

    8. nonuniExtern_ex0.py   - example #0 on how to use the sampler with the external sampling scheme

    9. nonuniExtern_ex1.py   - example #1 on how to use the sampler with the external sampling scheme

    10. nonuniExtern_ex2.py  - example #2 on how to use the sampler with the external sampling scheme


    # Uniform sampler examples:

    11. uniform_ex0.py - example #0 on how to use the uniform sampler - the simplest example


    # Saturation block examples:

    12. saturation_ex0.py        - example #0 on how to use the saturation block - the simplest example

    13. saturation_ex1.py        - example #1 on how to use the saturation block - the signal is uniformly sampled and

    13. saturation_ex2.py        - example #2 on how to use the saturation block
